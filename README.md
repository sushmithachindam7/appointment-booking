# 📱 Appointment Booking UI

A simple mobile-first appointment booking interface built with **HTML, CSS, and JavaScript**.  
This project demonstrates a basic booking flow for service-based businesses such as clinics, salons, consultants, or fitness trainers.

## ✨ Features
- Login / Onboarding screen  
- Service selection (Consultation, Fitness Training, Salon)  
- Date & time slot booking  
- Appointment confirmation screen
